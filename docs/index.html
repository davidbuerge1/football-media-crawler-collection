Crawler Projekt Uebersicht

Diese Seite zeigt die einzelnen Medien-Crawler, die Rohdaten und die Auswertungen. Die Tabellen sind als CSV gespeichert und in den jeweiligen README-Dateien dargestellt.

Wie der Crawler funktioniert

Ein Crawler durchsucht eine Website automatisch nach passenden Seiten. In diesem Projekt wird die Suche ueber Sitemaps gemacht.

Was ist eine Sitemap?

Eine Sitemap ist eine Datei (meist XML), die eine Liste von Seiten einer Website enthaelt. Sie hilft Programmen dabei, alle Inhalte zu finden.

Was macht der Crawler?

  • Er laedt die Sitemap(s) der jeweiligen Website.
  • Er filtert alle Fussball-Seiten.
  • Er speichert gefundene Links in einer URL-CSV.
  • Er zaehlt pro Jahr Frauen- und Herrenfussball.
  • Er schreibt die Jahreszahlen in eine Count-CSV.
